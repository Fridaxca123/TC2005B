Preguntas:

¿Cuál es la diferencia entre internet y la World Wide Web?

Mientras que internet es una red inmensa de computadoras conectadas entre si, la WWW es una enorme colección de paginas que se asienta sobre esa red de computadoras.

¿Cuáles son las partes de un URL?

- Un esquema: Protocolo para acceder al recurso em internet. HTTPS o HTTP

- Nombre host: www.example.com

- Un camino: Recurso especifico al que se quiere acceder. /software/htp/cisc/index.html

- Cadena de consulta

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

- GET: Solicitar datos de un recurso sin modificarlo.

- HEAD: Lo mismo que get pero solo se obtienen los encabezados sin el cuerpo.

- POST: Enviar datos para crear un nuevo recurso..

- PUT: Reemplazar completamente un recurso existente o crearlo si no existe.

- PATCH: Modificar parcialmente un recurso.

- DELETE: Eliminar un recurso.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

Se debe de usar POST para enviar un formulario HTML porque protege la información y permite enviar datos en el cuerpo de la solicitud en lugar de URL.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

Cuando accedes a una página web escribiendo un URL en el navegador, se usa el método GET porque solo estás solicitando información sin modificar nada en el servidor.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

El código 200 en una respuesta HTTP significa que la solicitud fue exitosa y el servidor devolvió el contenido solicitado sin errores.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Si el error es causado por el sitio web(archivos que se hayan eliminado o problemas de configuracion), si debe ser solucionado por el desarrollador. Sin embargo este problema tambien puede ser del usuario que el URL esta mal escrito.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Si porque un error 500 indica un problema en el servidor.

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)?

Un atributo depreciado significa que ya no se recomienda usarlo porque ya lo reemplazaron por otro mas moderno.

¿Cuáles son las principales diferencias entre HTML 4 y HTML 5?

En HTML algunas de las etiquetas se han simplificado por lo que es mas rapido y facil escribir codigo. Tambien se incorpora el uso de elementos dinamicos, ya no es tan necesario CSS y Javascript

¿Qué componentes de estructura y estilo tiene una tabla?

-Estructura: table, tr, th, td y caption

-Estilo: borders, border-colapse, text-align, padding y background-color

¿Cuáles son los principales controles de una forma HTML5?

Entre los principales controles se encuentran los campos de entrada (input), que incluyen tipos como text para texto libre, password para contraseñas, email, tel y url para datos específicos, number para valores numéricos y date, time y datetime-local para selección de fechas y horas.

¿Qué tanto soporte HTML5 tiene el navegador que utilizas?

HTML 5 es totalmente compatible con el navegador Google Chrome

¿Cuál es el ciclo de vida de sistemas de información?

Fase de planificacion, analisis, diseño, desarrollo, pruebas, implementacion y mantenimiento
