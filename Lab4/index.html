Preguntas:

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

El uso de !important es útil cuando se necesita sobreescribir estilos específicos en CSS. Sin embargo, solo se debe usar cuando sea necesario, de lo contrario, puede llevar a otras complicaciones.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Porque dependiendo del tamaño de píxeles de la imagen, la mayoría de veces estas suelen distorsionarse. Además, dificultan la visualización del texto escrito sobre la imagen y aumentan el tiempo de carga.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de una propiedad de estilo entre %, px y pt?

Esta decisión depende del diseño y el efecto que quieras lograr.

  • Porcentaje: Cuando se necesita que el elemento se ajuste proporcionalmente a un contenedor o al ancho de una pantalla.
  • Pixeles: Brinda un control más preciso del tamaño. No se ajusta a cambios de tamaño en contenedor o pantalla.
  • Puntos: Está diseñado para impresión tipo Word.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

El uso de una versión minimizada mejora el rendimiento porque:

  1. Reduce el tamaño del archivo.
  2. Mejora el tiempo de carga.
  3. Reducción de solicitudes HTTP.
  4. Mejora rendimiento en dispositivos móviles.
