Preguntas:

¿Que diferencias y semejanzas existen entre Java y Javascript?

La principal diferencia es que el proposito de Jaavascript esta orientado al desarrollo web, mientras que el de Java es más general. Mientras que Java es un lenguaje compilado que se debe ejecutar en una maquina virtual, Javascript se ejecuta directamente en un navegador web. Algunas de sus similitudes son que ambos son lenguajes orrinetados a objetos y utilizan sinytaxis similar (if, for. while, swich, {})

¿Que metodos tiene el objeto DATE?

EL objeto date se utiliza para manejar fechas y horas en Javascript los metodos más usados son:

  • getFullYear(): Devuelve el año
  • getMonth(): Devueleve el numero de mes
  • getDate(): Devuelve el dia del mes
  • getDay(): Devuel ve el dia de la semana en numero
  • toISOString(): Devuelve la fecha completa

¿Qué metodos tienen los arreglos?

Algunos de los metodos para arreglo son:

  • push(valos): Agrega al final de un arreglo
  • pop(): Elimina o devuelve el ultimo elemento
  • shift(): Elimina o devuelve el primer elemento
  • unshift(valor): Agrega un elemento al inicio

¿Cómo se declara una variable con alcance local dentro de una función?

Para declarar una variable con alcanace local se usa let y const.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Usar variables globales dentro de funciones puede parecer conveniente, pero introduce riesgos de mantenimiento, errores difíciles de depurar y posibles conflictos. La mejor práctica es usar variables locales o encapsularlas en funciones/módulos para mantener un código más seguro y predecible.
